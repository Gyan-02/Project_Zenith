# GYA-53 — Final Demo Script Tightening

Goal: create a tight 3–5 minute demo script that matches the actual product.

This should help whoever presents Project Zenith avoid wandering.

## Scope

### Modify

- `docs/demo-runbook.md`

### New optional file

- `docs/final-demo-script.md`

## Requirements

Write a presenter script with:

1. Opening line: what Project Zenith is.
2. Demo mode explanation in one sentence.
3. Sequence:
   - globe loads
   - search Saturn
   - narrator asks “Show me Saturn”
   - inspect object details
   - show conditions/events/passes
   - copy share link
   - mention offline fallback
4. Backup path if live narrator/API fails.
5. Closing line: why this matters.

## Acceptance checks

- Script fits in 3–5 minutes.
- Script references only features that exist in the app.
- No imaginary features.

## Boundaries

- Docs only.
- Do not change product code.
